# 3D Movement
A project to help learn how to render objects and create a player in 3-dimensional space. Graphics done with SDL.

## Usage
```
make
./3d_movement
```

## Controls
- WASD: Player movement
- IJKL: Camera movement
- Spacebar: Fly up
- Ctrl: Fly down
